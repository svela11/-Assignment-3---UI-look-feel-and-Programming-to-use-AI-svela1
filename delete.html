Remove student, course, or enrollment records from the system

How Delete Works

The Delete operation removes records from the database. In the final dynamic version, users will select a record and confirm its deletion. The system will send an API DELETE request to remove the corresponding row from the database.

Process: Select a record → Review details → Confirm deletion → API DELETE request is sent → Record is removed from the database → Confirmation message is shown.

Important: Deleting a record is permanent. The system will implement cascade rules to handle related records (e.g., deleting a student will also remove their enrollments).
