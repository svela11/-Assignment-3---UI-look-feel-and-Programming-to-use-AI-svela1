How Create Works

The Create operation allows administrators to add new records to the database. In the final dynamic version, this form will submit data via an API endpoint that inserts new rows into the corresponding database tables (Students, Courses, Enrollments, or Instructors).

Process: User fills out the form → Data is validated → API POST request is sent → New record is inserted into the database → Confirmation is displayed.
