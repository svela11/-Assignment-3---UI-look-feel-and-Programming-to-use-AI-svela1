How Read Works

The Read operation retrieves and displays data from the database. In the final project, this page will fetch data via API GET requests, displaying real-time records from the Students, Courses, and Enrollments tables.

Process: Page loads → API GET request is sent → Data is returned as JSON → Records are rendered in tables → Users can search and filter results.
